# 江南大学校园网信息爬虫

## 简介

这是针对江南大学校园网各网站的信息采集爬虫, 爬取各类校园网站信息入江南听雨数据库, 作为江南听雨其他产品的数据源.

这个爬虫是采用 [Python][py] 语言的 [Scrapy][scrapy] 框架开发的应用.

[py]: http://python.org/
[scrapy]: http://scrapy.org/


## 覆盖范围

* [江南大学教务处首页](http://jw.jiangnan.edu.cn/)上的文章链接
